How Product Owners & Business Can Use This Portal

For Product Owners / Business

  • Use Applications in Scope to understand where QA is engaged.
  • Use Core QA Assets to quickly open Test Strategy, DoR/DoD, and test data guidelines when refining stories.
  • Review Web/Mobile & API Automation to see what is already automated and where manual testing is still required.

For QA & Test Managers

  • Keep links updated for each tool, dashboard, and repository.
  • Update automation coverage and key metrics regularly so this portal remains the single source of truth.
  • Use the portal as a starting point in discussions with new stakeholders, vendors, and teams.
  • Capture new initiatives in the Roadmap and link to supporting documents.
